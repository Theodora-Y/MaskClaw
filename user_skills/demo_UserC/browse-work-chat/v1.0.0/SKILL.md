## 何时使用
在钉钉的工作群中浏览和回复消息时，该技能被触发。

## 执行步骤
- [ ] 步骤1：打开钉钉应用，进入目标工作群页面。
- [ ] 步骤2：长按需要回复的消息，选择“回复”选项。
- [ ] 步骤3：在弹出的输入框中输入回复内容，点击发送按钮完成回复。如果消息是@提及的，请先@相关昵称（如@张三），再输入回复内容。
- [ ] 步骤4：若需撤回已发送消息，点击消息右侧的“...”，选择“撤回”选项，并确认操作。注意：撤回操作仅在消息发送后1小时内有效。
- [ ] 步骤5：若遇到消息无法回复的情况，检查是否为系统通知或图片/文件消息（通常不可直接回复）。

## 边界情况
- [特殊情况1]：网络连接不稳定导致消息发送失败，提示“网络错误”。尝试重新发送或稍后重试。
- [特殊情况2]：用户未授权发送消息，提示“权限不足”。停止操作并通知用户获取授权。用户拒绝授权时，直接拒绝执行后续操作。
