## 何时使用
在微信的聊天中向好友转账时，该技能被触发。

## 执行步骤
- [ ] 步骤1：点击聊天界面中的“+”图标，选择“转账”选项。
- [ ] 步骤2：选择要转账的好友，确认对方账户信息无误后，点击“下一步”。
- [ ] 步骤3：输入转账金额，确保金额正确后，点击“完成”。
- [ ] 步骤4：在确认页面，再次核对转账金额和对方账户信息，点击“发送”完成转账操作。

## 边界情况
- [特殊情况1]：若用户在步骤2选择转账好友时发现对方账户信息有误，需立即返回重新选择好友或修改账户信息，避免错误转账。
- [特殊情况2]：若操作过程中出现网络中断或超时，提示用户检查网络连接并重新尝试操作；若用户拒绝继续操作，则终止当前转账流程。
