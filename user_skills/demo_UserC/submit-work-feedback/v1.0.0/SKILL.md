## 何时使用
在内部系统的在企业内部系统提交工作改进建议或投诉时，该技能被触发。

## 执行步骤
- [ ] 步骤1：进入内部系统，点击【工作改进建议/投诉】入口
- [ ] 步骤2：选择反馈类型（建议/投诉），点击【下一步】
- [ ] 步骤3：输入反馈标题，点击【下一步】
- [ ] 步骤4：详细描述问题或建议，确保内容真实、不泄露同事信息，点击【下一步】
- [ ] 步骤5：确认是否匿名反馈，若匿名需检查系统支持，点击【提交】
- [ ] 步骤6：核对所有信息无误后，点击【确认提交】，等待系统反馈
