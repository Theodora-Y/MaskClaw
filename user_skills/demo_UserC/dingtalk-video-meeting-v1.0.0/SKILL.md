## 何时使用
在钉钉的在钉钉中发起或加入一个视频会议时，该技能被触发。

## 执行步骤
- [ ] 步骤1：打开钉钉应用，点击底部导航栏的“工作”或“会议”图标（具体位置根据钉钉版本可能略有不同）。
- [ ] 步骤2：选择“发起会议”或“加入会议”，根据操作场景输入会议名称、密码或点击“加入”按钮。
- [ ] 步骤3：进入会议界面后，点击右上角的“更多选项”图标，选择“设置”或“隐私设置”。
- [ ] 步骤4：在隐私设置中，确保勾选“仅参会人员可见”和“禁止录音录像”，并关闭“自动分享屏幕”功能（如适用）。
- [ ] 步骤5：检查会议链接是否已生成，若已生成，请记下或保存，不要随意分享给无关人员。
- [ ] 步骤6：确认会议内容不涉及敏感信息，避免在公共场合接入会议，离开时退出会议界面。
