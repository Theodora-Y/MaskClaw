## 何时使用
在报销系统的在企业报销系统中提交差旅费用报销时，该技能被触发。

## 执行步骤
- [ ] 步骤1：点击报销系统首页的“提交差旅报销”按钮
- [ ] 步骤2：选择差旅类型（如国内出差），点击“下一步”
- [ ] 步骤3：上传差旅票据照片（确保清晰可见发票信息），点击“确认上传”
- [ ] 步骤4：填写差旅起止日期、出发地和目的地，点击“保存”
- [ ] 步骤5：填写同行人员信息（如有），点击“添加同行”
- [ ] 步骤6：核对所有填写信息无误后，点击“提交报销申请”，等待财务审核

## 边界情况
- [特殊情况1]：若上传票据模糊或缺失关键信息，提示重新上传并要求提供清晰的发票照片
- [特殊情况2]：若同行人员信息未按需填写，提示补充完整同行人员信息
- [特殊情况3]：若操作超时，自动返回报销首页并提示网络或服务器问题，建议稍后再试
- [特殊情况4]：若用户拒绝授权上传票据，提示需提供纸质票据备查，无法继续提交报销申请
