## 何时使用
在邮件的使用企业邮箱发送一封工作邮件时，该技能被触发。

## 执行步骤
- [ ] 步骤1：打开邮件应用，点击“新建邮件”按钮。
- [ ] 步骤2：输入收件人地址（仅填写脱敏后的邮箱地址），点击“收件人”栏下方的“添加联系人”按钮，选择“企业通讯录”中的相关联系人，并确认已脱敏。
- [ ] 步骤3：在正文区域输入邮件内容，确保未包含患者信息、内部通讯录或财务数据等敏感内容，使用AutoGLM的脱敏功能对敏感词汇进行处理。
- [ ] 步骤4：点击“发送”按钮，确认邮件已成功发送到目标邮箱地址，且内容符合隐私要求。

## 边界情况
- [特殊情况1]：如果邮件内容包含敏感信息，AutoGLM将自动提示用户进行内容审查，并建议使用mask或replace策略处理敏感字段。若用户拒绝修改，则执行block策略，阻止发送邮件。
- [特殊情况2]：网络连接失败或超时，AutoGLM将提示用户检查网络状态，若多次尝试仍失败，则执行deny策略，终止发送操作。
