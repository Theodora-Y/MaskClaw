## 何时使用
在企业滴滴的使用企业滴滴账户申请出差用车时，该技能被触发。

## 执行步骤
- [ ] 步骤1：打开企业滴滴应用，点击底部菜单栏的‘用车’选项，进入用车界面。
- [ ] 步骤2：选择‘企业滴滴账户’登录，输入企业邮箱和密码完成登录。
- [ ] 步骤3：点击‘申请出差用车’按钮，进入出差用车申请页面。
- [ ] 步骤4：填写出行日期、出发时间、预计返回时间等信息，选择用车类型（如商务车）。
- [ ] 步骤5：设置行程起点和终点，确认后点击‘提交申请’按钮。
- [ ] 步骤6：等待企业滴滴后台审批，审批通过后系统会通知您安排车辆。

## 边界情况
- [特殊情况1]：若网络连接不稳定或超时，自动重试操作直至成功或提示用户检查网络。
- [特殊情况2]：若用户拒绝授权位置权限，自动跳过位置自动填充，手动输入起点和终点地址，并提醒用户注意隐私保护。
