## 何时使用
在外卖平台的在美团或饿了么上下单点外卖时，该技能被触发。

## 执行步骤
- [ ] 步骤1：打开美团或饿了么应用，点击首页“点餐”按钮进入点餐页面。
- [ ] 步骤2：选择餐厅和菜品，添加到购物车，确认数量后点击“结算”按钮。
- [ ] 步骤3：在订单详情页，填写收货地址（仅用于配送），输入联系电话（仅用于配送），勾选同意服务协议后点击“提交订单”按钮。
- [ ] 步骤4：确认订单信息无误，点击“确认支付”完成下单。

## 边界情况
- [特殊情况1]：如果网络连接失败，提示用户检查网络并重新尝试。
- [特殊情况2]：如果用户拒绝提供收货地址或联系电话，提示用户这是必要信息，无法继续操作。
