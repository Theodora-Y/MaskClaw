## 何时使用

## 执行步骤
- [ ] 步骤1：打开手机上的邮件应用，点击“新建邮件”按钮。
- [ ] 步骤2：在收件人栏输入企业邮箱地址，点击“发送”按钮前，检查邮件内容是否包含患者信息、内部通讯录或财务数据等敏感内容。
