**步骤 2: 进入聊天列表**

异常处理：若界面无反应，尝试滑动屏幕解锁或重启应用；若提示网络中断，重新检查网络连接；若应用闪退，点击“强制停止”并重新打开  

**步骤 3: 选择聊天对象**  
具体动作：在聊天列表中点击目标联系人的头像或昵称，进入聊天窗口  
异常处理：若联系人不在列表中，点击右上角的“+”按钮，搜索联系人；若应用崩溃，点击“强制停止”并重新打开；若权限不足导致无法访问联系人列表，返回上一步开启权限后重试  

**步骤 4: 输入消息内容**  
具体动作：在聊天输入框中长按，选择“表情”或“图片”选项；或直接在输入框中输入文本内容  
异常处理：若输入框无法使用，返回上一界面检查网络状态或应用权限设置；若网络中断，尝试重新连接网络；若权限不足（如缺少存储权限），提示用户开启相关权限后重试  

**步骤 5: 发送消息**  
具体动作：点击输入框右侧的“发送”按钮（或键盘上的回车键），发送消息  
明确处理：若消息发送失败且网络正常，建议：  
- 检查微信是否需要更新，前往应用商店检查更新；  
- 尝试重新启动微信应用；  
- 若多次尝试仍失败，可尝试联系微信客服寻求帮助；  
- 若应用闪退后多次操作仍失败，建议清理手机缓存或重新启动手机；  
- 若问题持续，可尝试卸载并重新安装微信应用；  
自动重试：若发送失败，系统将自动尝试重新发送消息，最多重试 3 次，若仍失败则提示用户手动检查网络或应用状态。  

**步骤 6: 检查消息发送状态**  
具体动作：观察聊天窗口中消息气泡的颜色变化，确认消息是否成功发送  
异常处理：若消息气泡显示为灰色，可能因网络问题未能发送，尝试重新发送；若应用崩溃，点击“强制停止”并重新打开  

**步骤 7: 返回主界面**  
具体动作：点击聊天窗口右上角的“返回”按钮，回到微信主界面  
异常处理：若返回失败，点击手机顶部的“返回键”退出当前界面；若网络中断，检查并重新连接网络
