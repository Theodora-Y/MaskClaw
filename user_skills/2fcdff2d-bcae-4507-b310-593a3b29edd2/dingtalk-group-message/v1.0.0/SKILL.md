# 钉钉工作群发消息 — 隐私保护规范

## 应用场景
在钉钉的工作群聊中发送一条消息。

## 隐私保护规则
工作群消息仅限群内可见，不得转发至外部；消息内容不得包含未脱敏的客户信息。

## 执行步骤

- [ ] **步骤 1：内容合规检查** — 检查消息是否包含客户信息、财务数据、内部通讯录等敏感内容
- [ ] **步骤 2：脱敏处理** — 如有敏感信息，建议使用代号或脱敏后内容
- [ ] **步骤 3：确认发送** — 确认接收人范围正确后执行发送

## 隐私约束

- 发送前检测是否包含客户姓名、电话、地址等客户信息
- 财务相关数据（金额、账户）发送前需脱敏
- 转发至外部联系人前强制确认

## 边界情况

- 紧急通知场景优先效率，但事后需检查隐私合规性
- 跨部门协作群聊需额外确认信息来源和必要性
