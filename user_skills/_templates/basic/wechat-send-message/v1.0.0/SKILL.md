# 发送微信消息 — 隐私保护规范

## 应用场景
在微信聊天窗口中发送一条文字消息。

## 隐私保护规则
消息内容可能被监控或转发，需注意不得包含未脱敏的病历、身份证、家庭住址等隐私信息。

## 执行步骤

- [ ] **步骤 1：内容审查** — 检查消息内容是否包含身份证号、家庭住址、手机号、病历等隐私信息
- [ ] **步骤 2：隐私提醒** — 如有敏感内容，弹出提醒让用户确认是否发送
- [ ] **步骤 3：发送** — 用户确认后执行发送

## 隐私约束

- 发送前自动检测是否包含身份证号、家庭住址等敏感信息
- 医疗相关词汇出现时提示用户确认
- 转发消息时提示原始内容隐私风险

## 边界情况

- 紧急情况下需权衡隐私和效率
- 家庭群和工作群适用不同隐私级别
