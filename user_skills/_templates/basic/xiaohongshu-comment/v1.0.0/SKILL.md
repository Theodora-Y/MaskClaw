# 发送小红书评论 — 隐私保护规范

## 应用场景
在小红书帖子下方发布一条评论。

## 隐私保护规则
评论内容公开可见，不得包含真实姓名、手机号、就诊记录等隐私信息。

## 执行步骤

- [ ] **步骤 1：隐私内容检测** — 检查评论内容是否包含手机号、姓名、身份证、就诊医院等可直接识别的隐私信息
- [ ] **步骤 2：内容脱敏** — 如有敏感信息，建议用户使用昵称或模糊描述替代
- [ ] **步骤 3：发布确认** — 用户确认内容适合公开后执行发布

## 隐私约束

- 评论前自动检测是否包含手机号、身份证等可直接识别的隐私信息
- 医疗健康相关话题评论时不得暴露就诊经历
- 涉及工作内容时确认是否适合公开

## 边界情况

- 涉及投诉举报场景时需额外确认信息准确性
- 与陌生人互动时谨慎透露任何可定位信息
